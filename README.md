# tunnel
Simple tunnel manager. Supports bash and zsh.


## Installation

### Manually
```bash
git clone --recursive https://github.com/tweetgeek/tunnel
```
then in .bashrc or .zshrc:
```bash
source ./tunnel/tunnel.sh
```
